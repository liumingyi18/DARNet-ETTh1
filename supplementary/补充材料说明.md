# 补充材料说明

本目录用于保存不属于统一六模式主实验、但有助于理解项目演进和优化过程的材料。

- `0114-DARNet-V4-finaltry.py`：基于正式实验版继续进行的最后一轮优化代码。
- `task1_optimization_log.csv`：补充优化每轮参数和指标。
- `all_finaltry_results_summary.csv`：补充实验汇总。
- `round5_patch_huber.csv`：补充实验中表现最好的 Patch + Huber 结果。
- `round5_patch_huber.pt`：上述实验对应的模型权重。
- `source_history/`：初始代码、统一修正版和 Notebook。

论文六种模式的正式复现入口仍是：

```text
../code/0114-DARNet-V4-opt.py
```

论文六模式统一结果仍以：

```text
../results/all_modes_summary.csv
```

为准。
