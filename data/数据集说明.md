# ETTh1 数据集说明

本归档使用 `ETTh1.csv`，共 17,420 条小时级记录。

- 原始数据集：[zhouhaoyi/ETDataset](https://github.com/zhouhaoyi/ETDataset)
- 对应文件：`ETT-small/ETTh1.csv`
- 原始许可证：[CC BY-ND 4.0](https://creativecommons.org/licenses/by-nd/4.0/)
- 相关论文：[Informer: Beyond Efficient Transformer for Long Sequence Time-Series Forecasting](https://arxiv.org/abs/2012.07436)

本仓库中的文件与原始 ETTh1 文件 MD5 一致。公开发布或再分发时必须保留来源和许可证说明，不得以修改后的数据替代该原始文件并继续沿用同一授权声明。

- 时间范围：2016-07-01 00:00:00 至 2018-06-26 19:00:00
- 时间字段：`date`
- 输入与目标字段：`HUFL`、`HULL`、`MUFL`、`MULL`、`LUFL`、`LULL`、`OT`
- 预测目标：变压器油温 `OT`
- 输入窗口：过去 72 小时
- 预测窗口：未来 24 小时

字段含义：

| 字段 | 中文含义 |
|---|---|
| date | 时间戳 |
| HUFL | 高压负荷下界 |
| HULL | 高压负荷上界 |
| MUFL | 中压负荷下界 |
| MULL | 中压负荷上界 |
| LUFL | 低压负荷下界 |
| LULL | 低压负荷上界 |
| OT | 变压器油温 |

程序按时间顺序将前 80% 数据作为训练及验证部分，后 20% 作为测试部分。归一化器只在前 80% 数据上拟合，避免测试集信息泄漏。
